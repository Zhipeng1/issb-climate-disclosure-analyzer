# 行业模块：电力、公用事业与新能源运营商（Power & Utilities）

> 适用对象：火电运营商、可再生能源运营商、综合电力公司、输配电企业、公用事业集团
> S2 最典型的高影响行业之一，适合测试转型计划质量

---

## 一、本行业最重要的气候风险与机遇

### 转型风险（一阶）
- 碳价直入发电成本：火电 Scope 1 × 碳价 → 发电成本上升，直接影响上网电价竞争力
- 资产搁浅：煤电/气电资产在低碳情景下面临提前退役和减值，剩余使用寿命缩短
- 政策与法规风险：碳交易/碳税、可再生能源配额、煤电退出时间表、容量电价改革
- 制度前提（评估任何条款前先确认）：受规管（RAB/准许回报/燃料调整账户）vs 市场化业务，同一块钱低碳 Capex 在两种机制下财务命运完全不同；同一集团内两种机制并存时须分部评估，不做集团平均

### 物理风险
- 发电效率影响：高温降低火电和核电冷却效率、干旱影响水电出力
- 电网韧性：极端天气（冰灾/风暴/高温）导致输配电设施损坏和大面积停电
- 可再生能源波动：气候模式变化影响风光资源的可预测性

### 气候机遇
- 可再生能源装机增长：风电/光伏/储能投资的长期增长空间
- 电网升级：智能电网、储能、分布式能源的投资需求
- 绿色电力产品：绿电 PPA、绿证、24/7 清洁电力服务的溢价

## 二、最容易触发的 S2 条款

| 条款 ID | 条款 | 电力行业触发原因 |
|---|---|---|
| ID-059/060 | §29(a)(i)(1-2) Scope 1+2 | 火电排放量大，是碳成本的核心输入 |
| ID-068 | §29(b) 转型风险金额和% | 煤电/气电资产的账面价值和搁浅风险 |
| ID-071 | §29(e) 气候 Capex | 新能源/储能/电网升级的投资规模 |

| ID-033/038 | §22 情景分析 | 不同碳价/政策情景下的资产组合韧性 |
| ID-077/081 | §33(b)(f) 目标与里程碑 | 排放强度下降路径和煤电退出时间表 |

## 三、最关键的投资有用指标

| 指标 | 投资模型变量 | 投资者关注重点 |
|---|---|---|
| 排放强度（tCO2e/MWh） | COGS/碳成本 | 发电组合碳强度及下降趋势，对标行业脱碳路径 |
| 发电结构占比（煤/气/核/水/风光） | 收入结构/资产质量 | 化石能源 vs 可再生能源的装机和发电量占比 |
| 煤电退出时间表 | 资产减值/搁浅 | 各煤电资产的计划退役年份和剩余账面价值 |
| 新能源 Capex 占总 Capex 比例 | 再投资率 | 资本配置是否向可再生能源倾斜 |
| 碳价敏感性（EBITDA 弹性） | 盈利波动 | 碳价每上升 $10 对集团利润的百分比影响 |
| 资产搁浅风险敞口（金额） | EV/资产减值 | 在 1.5°C 情景下需提前退役的资产账面价值 |

## 四、常见误判

| 误判模式 | 正确判定 | 适用 ID |
|---|---|---|
| 新能源装机增长 = 集团排放强度下降 | 如果煤电不退出，新增装机只稀释占比但绝对排放可能不降 | ID-059/060 |
| 披露减排目标 = 披露煤电退出路径 | 减排目标是终态，煤电退出路径需要逐电厂的退役年份和替代方案 | ID-015/019 |
| 披露装机结构 = 披露资产搁浅风险 | 装机数据是实物量，搁浅风险需要资产账面价值和减值测试 | ID-068/023 |
| 碳排放权交易描述 = 碳价敏感性分析 | 参与碳市场是交易活动，S2 要求的是碳价变化对财务的影响量化 | ID-068/072 |
| 发布可再生能源投资计划 = 披露气候 Capex | 可再生能源投资是业务扩张，S2 §29(e) 要求的是"投向气候的 Capex"口径 | ID-071 |
| 强制碳定价覆盖容量占比小 = 转型风险小 | 显性碳价敞口 ≠ 全部高碳容量面临的市场化、调度下降、减值与退役风险 | ID-068 |
| 权益容量百分比/风险等级 = §29(b)/(c) 的资产金额和% | 容量百分比与风险分级是 proxy_metric，非账面金额口径 | ID-068/069 |
| CO2 避免量（对反事实基准）= 自身减排 | 避免量依赖基准假设，不改变账面排放绝对值，不能替代 §35 进展 | ID-088 |
| 关闭成本拨备入账 = 政策突变损失 | 拨备源于推定义务的确认时点（关闭时间/成本被确定），须读触发说明再归因 | ID-023 |
| 受规管回报机制 = 股东无转型风险 | 电价承受力、政策目标与监管周期审批都可约束低碳 Capex 入基和回收 | ID-015/023 |

## 五、推荐 best-practice 做法类型

| 做法类型 | 说明 | 典型来源 |
|---|---|---|
| 资产组合情景分析 | 按资产级别做多情景下的估值和退役分析 | CLP Climate Vision 2050 |
| 煤电退出逐电厂时间表 | 列出每个煤电资产的计划退役年份和替代方案 | CLP |
| 排放强度对标行业路径 | 将自身排放强度与 IEA/SBTi 行业脱碳路径对比 | 待补充 |
| 碳价敏感性量化 | 按不同碳价情景量化对 EBITDA/净利润的百分比影响 | 待补充 |
| 新能源/储能 Capex 分项 | 按技术类型（风/光/储能/电网）拆分 Capex 金额和时间表 | 待补充 |
| 资产搁浅减值测试 | 在 1.5°C 情景下对煤电资产做减值敏感性分析 | BHP（矿业类比） |
| 气候融资用途逐笔披露+外部鉴证 | 转型/绿色融资交易逐笔列示用途、分配与影响指标，经独立有限鉴证 | CLP Climate Action Finance Report |
| 物理风险分灾种量化 | 按灾种披露高暴露容量比例及已确认/预期财务影响，未量化灾种说明原因 | CLP |
