# 行业模块：消费品、食品与农业供应链（Consumer Goods, Agri-food）

> 适用对象：快消品（FMCG）、食品饮料、日化、农产品贸易商、零售商
> Scope 3 和 FLAG 排放代表行业——供应链碳管理复杂度高

---

## 一、本行业最重要的气候风险与机遇

### 转型风险
- Scope 3 主导：供应链排放（原材料采购、农业、包装、物流）占总排放 90%+，自身运营排放极小
- FLAG 排放：食品/土地/农业（FLAG）排放来源复杂（甲烷、N2O、土地利用变化），脱碳路径与工业排放本质不同
- 包装法规：塑料税、EPR（延伸生产者责任）、再生材料配额增加成本
- 森林砍伐合规：EU EUDR（反森林砍伐法规）影响大豆、棕榈油、牛肉、可可、咖啡等原材料采购

### 物理风险
- 农业产区气候影响：干旱/洪涝/高温影响原材料产量和价格（可可、咖啡、小麦、牛奶）
- 水资源约束：食品加工和农业灌溉依赖水资源，缺水地区面临供应中断
- 供应链地理集中：关键原材料产区集中在气候脆弱地区

### 气候机遇
- 低碳/植物基产品：植物蛋白、有机产品、低碳足迹产品线的增长空间
- 可持续包装：再生材料、可降解包装、减塑方案
- 供应链韧性投资：供应商多元化和气候适应型农业投资

## 三、最关键的投资有用指标

| 指标 | 投资模型变量 | 投资者关注重点 |
|---|---|---|
| Scope 3 按 FLAG/工业分组 | COGS 结构 | FLAG 排放 vs 工业排放的占比及各自的脱碳难度 |
| 供应商 PCF 数据覆盖率 | 供应链碳管理成熟度 | 已收集产品碳足迹数据的供应商比例 |
| 森林砍伐零净值承诺进展 | 合规/市场准入 | 关键商品（棕榈油/大豆/牛肉/纸浆）的可追溯性 |
| 低碳产品收入占比 | 收入结构 | 植物基/有机/低碳足迹产品在总收入中的份额和增速 |
| 包装可回收/再生材料占比 | 合规成本 | 塑料税和 EPR 法规下的成本影响 |
| 关键原材料产区物理风险暴露 | 供应连续性/COGS | 高风险产区占采购量的百分比 |

## 四、常见误判

| 误判模式 | 正确判定 | 适用 ID |
|---|---|---|
| 供应商倡议/承诺 = Scope 3 实际下降 | 倡议是过程，需检查 Scope 3 绝对量是否逐年下降 | ID-061 |
| 产品环保宣传 = 气候机遇收入 | "绿色""天然""环保"标签不等于 §29(d) 要求的"与气候机遇对齐的收入金额" | ID-070 |
| 不使用碳信用承诺 = 已披露完整净零路径 | 不依赖碳信用是正面信号，但仍需检查自身减排路径是否有具体措施和里程碑 | ID-093/015 |
| 可持续采购政策 = FLAG 排放量化 | 采购政策是过程，FLAG 排放需要按 SBTi FLAG 方法论量化 | ID-066 |
| 供应商行为准则含气候 = 供应链碳管理工具化 | 行为准则是要求，工具化需要数字平台、PCF 数据收集和排放追踪 | ID-017 |
| 森林砍伐承诺 = 已实现可追溯 | 需检查关键商品的可追溯比例和第三方认证覆盖 | 相关性判断 |

## 五、推荐 best-practice 做法类型

| 做法类型 | 说明 | 典型来源 |
|---|---|---|
| Scope 3 FLAG/工业分组目标 | 分别设定 SBTi 验证的 FLAG 和工业排放目标 | Unilever |
| 供应商 PCF 纳入采购评分 | 将产品碳足迹数据作为采购决策的评分因子 | Unilever |
| 近期目标不依赖碳信用 | 明确 2030 前减排目标全部通过自身和供应链减排实现 | Unilever |
| 供应链脱碳数字化平台 | 通过数字工具追踪供应商排放和减排进展 | 施耐德电气 Zeigo Hub |
| 绿色收入占比量化 | 量化可持续产品在总收入中的占比并逐年追踪 | 施耐德电气 Impact Revenues |
| SBTi FLAG 方法论 | 按 SBTi FLAG 标准单独量化食品/土地/农业排放 | Unilever |
