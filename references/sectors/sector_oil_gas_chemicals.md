# 行业模块：油气与化工（Oil, Gas & Chemicals）

> 适用对象：上游勘探开发（E&P）、炼化一体化、天然气运营商、石化企业、特种化学品
> S2 最复杂的行业之一——Scope 3 下游使用、资产搁浅、碳捕集、甲烷争议集中

---

## 一、本行业最重要的气候风险与机遇

### 转型风险（一阶）
- Scope 3 下游使用排放：油气企业最核心争议——产品燃烧排放占总排放 80%+，决定商业模式的长期存续性
- 资产搁浅：在低碳情景下未开发储量和高成本产能面临搁浅，需下调储量估值和长期价格假设
- 碳价与法规：碳交易/碳税直接影响炼化成本，甲烷排放法规（EU/US）增加合规成本
- 资本配置争议：化石能源 vs 低碳业务的投资比例是转型可信度的核心指标

### 物理风险
- 海上平台和沿海设施面临海平面上升和极端风暴
- 管网和炼厂受极端天气中断影响
- 水资源约束影响页岩气压裂和炼化冷却

### 气候机遇
- CCUS（碳捕集、利用与封存）：新技术和碳价上升可能创造碳管理收入
- 产品转型：生物燃料、氢能、低碳化学品、回收材料
- 碳管理服务：为下游客户提供碳管理/碳捕集服务

## 三、最关键的投资有用指标

| 指标 | 投资模型变量 | 投资者关注重点 |
|---|---|---|
| Scope 3 下游使用排放绝对量 | 商业模式存续性 | 产品燃烧排放量及下降趋势，对标 IEA 需求路径 |
| 低碳投资 vs 化石能源投资比例 | 资本配置/再投资率 | 低碳 Capex 占总 Capex 的百分比和趋势 |
| 甲烷排放强度（kg CH4/boe） | 合规成本/声誉 | 甲烷泄漏和燃除的绝对量和强度 |
| 储量减值敏感性 | 资产减值/EV | 不同油价/碳价假设下的储量净现值变化 |
| CCUS 捕集量与成本 | 技术可信度 | 实际捕集量 vs 目标、单位成本、永久封存率 |
| 情景对标结果 | 终值增长/资产价值 | IEA NZE vs STEPS 情景下的产量路径和资产价值差异 |

## 四、常见误判

| 误判模式 | 正确判定 | 适用 ID |
|---|---|---|
| 运营排放（Scope 1+2）下降 = 商业模式转型 | Scope 1+2 是运营端，商业模式由 Scope 3 下游使用排放决定 | ID-061 |
| CCUS 项目公告 = 已形成可靠减排贡献 | 需检查实际捕集量、成本、永久性和技术成熟度 | ID-093/094 |
| 低碳投资金额 = 资本配置与净零路径一致 | 需看低碳投资占总 Capex 比例，以及化石能源端是否仍在扩张 | ID-071/015 |
| 自愿减排目标覆盖 Scope 1+2 = 有效气候目标 | 油气企业不覆盖 Scope 3 的目标回避了核心问题 | ID-090 |
| 披露储量寿命 = 披露搁浅风险 | 储量寿命是实物指标，搁浅风险需要在低碳情景下的减值测试 | ID-023/068 |
| 天然气定位为"转型燃料"= 低碳路径 | 天然气仍是化石能源，需在情景分析中验证其长期角色 | ID-033/038 |
