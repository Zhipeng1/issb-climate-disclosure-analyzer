# 行业模块：金融机构（Financial Institutions）

> 适用对象：商业银行、投资银行、保险公司、资产管理公司、开发性金融机构
> IFRS S2 对金融业有特殊披露压力，尤其是融资排放（Scope 3 Category 15）

---

## 一、本行业最重要的气候风险与机遇

### 转型风险（一阶）
- 融资排放（Financed Emissions）：贷款/投资/承保组合的碳排放是银行气候暴露的核心——客户的转型风险通过信用风险传导至银行
- 高碳行业敞口集中度：电力、油气、钢铁、水泥、航运等高碳行业贷款占比决定了转型风险的大小
- 资产搁浅传导：被投企业/借款人的高碳资产搁浅 → 信用恶化 → 银行减值损失
- 碳密集行业贷款定价：未来碳价上升 → 借款人偿债能力下降 → 信用利差扩大

### 物理风险
- 抵押品价值受损：房地产贷款中位于洪水/海平面上升/极端天气高风险区域的抵押品面临贬值
- 保险承保风险：极端天气事件频率和严重性上升 → 赔付增加 → 承保利润率下降
- 客户运营中断：借款人因物理风险导致运营中断 → 还款能力下降

### 气候机遇
- 可持续金融：绿色贷款、绿色债券、转型金融、可持续挂钩贷款（SLL）的增长
- 气候咨询服务：帮助客户制定转型计划、碳管理方案的咨询费收入
- 绿色溢价（greenium）：绿色债券发行中的定价优势

## 二、最容易触发的 S2 条款

| 条款 ID | 条款 | 金融业触发原因 |
|---|---|---|
| ID-067 | §29(a)(vi)(2) 融资排放 | 金融业专属——贷款/投资/承保组合的排放是核心指标 |
| ID-066 | §29(a)(vi)(1) Scope 3 类别拆分 | 需按 GHG Protocol Category 15 拆分融资排放 |
| ID-068 | §29(b) 转型风险金额和% | 高碳行业贷款/投资敞口的账面价值 |
| ID-033/038 | §22 情景分析 | 气候压力测试（NGFS 情景）是监管要求 |
| ID-058 | §25(c) 风管整合 | 气候风险嵌入信贷审批流程 |
| ID-015/019 | §14(a) 转型计划 | 组合脱碳目标和路径 |
| ID-023 | §16(b) 资产减值 | 高碳敞口的预期信用损失（ECL）影响 |

## 三、最关键的投资有用指标

| 指标 | 投资模型变量 | 投资者关注重点 |
|---|---|---|
| 融资排放（按行业拆分，MtCO2e） | 信用风险暴露 | 电力/油气/钢铁/航运/地产等行业各自的融资排放量和占比 |
| 组合碳强度（tCO2e/百万美元贷款） | 资产质量 | 贷款组合整体碳强度及下降趋势 |
| 高碳行业敞口占比 | 集中度风险 | 高碳行业贷款/投资/承保在总敞口中的金额和百分比 |
| 气候压力测试结果（ECL 增量） | 资本充足 | NGFS 情景下预期信用损失增量和资本影响 |
| 可持续金融余额及增速 | 收入增长 | 绿色贷款/绿色债券/SLL 的规模、增速和利差 |
| 客户转型计划评估覆盖率 | 风管质量 | 已评估客户转型计划可信度的高碳敞口比例 |

## 四、常见误判

| 误判模式 | 正确判定 | 适用 ID |
|---|---|---|
| 绿色金融余额增长 = 融资排放下降 | 绿色金融是增量，不代表存量高碳敞口减少；需看总组合碳强度变化 | ID-067 |
| 设定净零承诺 = 有行业组合脱碳路径 | 净零承诺是声明，需检查是否按行业设定了具体的碳强度下降目标和时间表 | ID-019/077 |
| ESG 风险纳入信贷审批 = 已量化气候信用风险 | "纳入考虑"是流程描述，需检查是否有量化的气候压力测试结果（ECL 增量） | ID-058/053 |
| 赤道原则签署 = S2 风管整合 | 赤道原则覆盖项目融资，不覆盖全部贷款组合的气候风管 | ID-058 |
| TCFD 报告完整 = S2 充分满足 | 金融业 TCFD 报告通常缺少 §29(b)-(e) 的金额和百分比 | 全部 |
| 自身运营碳中和 = 气候风险管理到位 | 银行自身运营排放极小（Scope 1+2），核心是融资排放（Scope 3 Cat 15） | ID-067 |

## 五、推荐 best-practice 做法类型

| 做法类型 | 说明 | 典型来源 |
|---|---|---|
| 融资排放分行业量化 | 按 7+ 行业披露融资排放量，使用 PCAF 方法论 | DBS（7 行业拆分） |
| 行业组合脱碳目标 | 按高碳行业设定碳强度下降目标和时间表 | DBS |
| 气候嵌入信贷审批 | 高碳行业贷款审批需通过气候风险评估门槛 | DBS |
| NGFS 气候压力测试 | 按 NGFS 情景运行信贷组合压力测试，披露 ECL 增量 | 待补充 |
