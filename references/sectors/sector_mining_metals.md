# 行业模块：矿业与金属（Mining & Metals）

> 适用对象：金属矿业（铜、金、铁矿石、锌、铝、锂等）、煤矿、矿业综合企业
> 紫金矿业测试已验证本模块核心判断

---

## 一、本行业最重要的气候风险与机遇

### 转型风险（一阶）
- 碳价直入 COGS：Scope 1+2 × 碳价 → 冶炼/采选成本直接上升
- 碳边境调节（CBAM）：冶炼产品出口至欧盟面临碳关税，影响市场准入和产品竞争力
- 矿石品位下降：品位持续下降推高单位能耗和排放强度，碳成本压力随品位递减而放大

### 物理风险（一阶）
- 资产级物理暴露：矿山/尾矿库面临洪涝、滑坡、高温、水压力等风险，且不可迁移
- 水资源约束：采选/冶炼高度依赖水资源，缺水地区矿山面临运营中断
- 极端天气导致供应链中断：运输通道（铁路/港口）受灾影响产品交付

### 气候机遇
- 转型金属需求增长：铜（电气化）、锂（储能）、钴/镍（电池）需求受气候政策驱动
- 绿色溢价：低碳金属产品在下游客户供应链脱碳中获得溢价和优先采购

## 二、最容易触发的 S2 条款

| 条款 ID | 条款 | 矿业触发原因 |
|---|---|---|
| ID-059/060 | §29(a)(i)(1-2) Scope 1+2 | 采选/冶炼排放量大，是碳成本的基础数据 |
| ID-068 | §29(b) 转型风险金额和% | 碳密集型资产（冶炼厂、煤矿）是搁浅风险核心 |
| ID-069 | §29(c) 物理风险金额和% | 矿山不可迁移，物理暴露直接关联资产价值 |
| ID-071 | §29(e) 气候 Capex | 绿电改造、矿卡电动化、CCUS 投资规模大 |
| ID-023 | §16(b) 资产减值 | 高碳资产在低碳情景下的减值是核心估值问题 |
| ID-072/073 | §29(f) 内部碳价 | 矿业项目 NPV 对碳价高度敏感 |
| ID-033 至 ID-037 | §22(a) 韧性结论 | 长寿命资产需证明多情景下的韧性 |

## 三、最关键的投资有用指标

| 指标 | 投资模型变量 | 投资者关注重点 |
|---|---|---|
| 产品碳强度（tCO2e/t 金属） | COGS/碳成本 | 不同金属、不同矿山的单位碳成本差异 |
| 碳价敏感性（EBITDA 对 $X/tCO2e 的弹性） | 盈利波动/WACC | 碳价每上升 $10 对利润的百分比影响 |
| 脱碳 Capex 总额及分项 | 再投资率 | 绿电/电动设备/CCUS/余热回收的投资规模和时间表 |
| 受物理风险影响的资产账面价值 | 资产减值 | 高风险矿山/尾矿库对应的资产金额 |
| 转型金属收入占比 | 收入结构 | 铜/锂等"绿色金属"在总收入中的份额和增长趋势 |
| 水资源强度（m³/t 矿石） | 运营连续性 | 缺水地区矿山的水风险及替代水源成本 |

## 四、常见误判

| 误判模式 | 正确判定 | 适用 ID |
|---|---|---|
| 矿山风险等级（高/中/低）= 资产金额 | 风险等级是定性分级，非 §29(c) 要求的"金额和%"| ID-069 |
| 绿色金属需求增长描述 = 气候机遇财务影响 | 需求增长是行业趋势，非企业自身"与机遇对齐的收入金额"| ID-070 |
| 未来建设新能源项目 = 已披露转型 Capex | 计划投资 ≠ 已投资/已量化的气候 Capex 总额 | ID-071 |
| 矿山复绿/生态修复支出 = 气候资本开支 | 生态修复是环境合规，非 S2 定义的气候转型投资 | ID-071 |
| 产品碳强度逐年下降 = 绝对排放下降 | 产量增长可能导致绝对排放不降反升 | ID-059/060 |
| 尾矿库安全管理 = 物理风险披露 | 尾矿库安全是矿业风管常规，S2 要求的是气候驱动的物理风险及其财务影响 | ID-069 |

## 五、推荐 best-practice 做法类型

| 做法类型 | 说明 | 典型来源 |
|---|---|---|
| 逐资产脱碳 Capex 量化 | 在财报附注中按运营资产列明脱碳投资金额和时间表 | BHP Note 16 |
| 1.5°C 情景减值敏感性 | 对主要资产做 1.5°C 情景下的减值测试并逐资产披露 | BHP |
| 产品碳强度逐年 tracking | 按金属产品线披露碳强度（tCO2e/t）并逐年对比 | 紫金矿业 |
| 影子碳价嵌入项目 NPV | 所有新项目投资决策须在碳价情景下仍有正 NPV | BHP |
| 气候投入总额独立披露 | 年度气候相关投入总额单独列示（含绿电/电动设备/碳技术） | 紫金矿业 |
| 矿山级物理风险量化 | 按矿山/资产给出物理风险暴露的资产价值和风险参数 | 待补充 |
| 水风险按矿山定位 | 按矿山所在地的水压力等级和用水强度披露水风险 | 待补充 |
