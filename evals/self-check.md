## 合规评估自检
- [ ] 四大支柱（治理/战略/风管/指标目标）全部覆盖，无遗漏支柱
- [ ] 每个子款(i)(ii)(a)(b)独立评分，与 s2_evaluation_framework 的 ID-001 至 ID-096 对齐
- [ ] **96 条全部列出**，不做"精选"，不遗漏任何子款（运行 clause_count_test）
- [ ] 合规大表含 **ID 列**（ID-001 至 ID-096），每行有唯一 ID
- [ ] 合规大表含 **证据页码列**

- [ ] §29(b)-(d)：已核实企业给的是"金额和百分比"还是替代指标（scoring_edge_cases 规则 4）
- [ ] §29(d)：已区分"避免排放量/绿色项目描述"和"机遇对齐的收入/资产金额"（scoring_edge_cases 规则 9）

- [ ] "企业实际披露"列写了具体数据和做法摘录，不是空泛描述

- [ ] "差距/改进建议"列：每条 ◐ 或 ○ 评分的子款都有具体改进方向，不可留空或仅写"——"
- [ ] "差距/改进建议"列引用了 best_practices 中的做法（不点名比较对象）
